# futsal-booking
Sistem Booking Lapangan Futsal

Sistem Booking Lapangan Futsal adalah aplikasi berbasis web yang dirancang untuk mempermudah proses pemesanan lapangan futsal secara digital. Sistem ini menggantikan proses pencatatan manual yang rentan terhadap kesalahan dan ketidakefisienan, dengan antarmuka yang mudah digunakan oleh pengelola

Fitur Utama
1. Manajemen Lapangan
-Menambah data lapangan baru (nama & harga per jam).
-Mengubah informasi lapangan yang sudah ada.
-Menghapus data lapangan.

2. Manajemen Booking
-Mencatat pemesanan lapangan oleh pengelola.
-Input data: nama customer, lapangan, tanggal, jam mulai, dan jam selesai.
-Kalkulasi total harga otomatis berdasarkan durasi booking.
-Mengubah dan menghapus data booking.

Teknologi yang Digunakan
Backend: PHP — Laravel Framework
Autentikasi: Laravel Breeze
Frontend: HTML, CSS, JavaScript (jQuery)
UI Framework: Bootstrap 5
Database: MySQL
Templating: Blade (Laravel)
AJAX: jQuery AJAX
